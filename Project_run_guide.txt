------------------------------------------------------------
1. ZForce FaceService (Python / FastAPI)
------------------------------------------------------------
Open PowerShell:

cd "D:\Zahir-Boss-Task\Project ZForce\ZForce-FaceService"

# (First time only) create venv + install deps
py -m venv .venv
.\.venv\Scripts\Activate.
py -m pip install --upgrade pip
py -m pip install -r requirements.txt

# If you ever get tf_keras error:
py -m pip install tf-keras

# Run server
py -m uvicorn main:app --host 0.0.0.0 --port 7001 --reload

Health check:
http://localhost:7001/health
